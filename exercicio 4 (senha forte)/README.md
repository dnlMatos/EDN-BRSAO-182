# Projeto: Verificador de Senha Forte

Este projeto é um programa em Python que verifica se uma senha é forte.

## Regras para senha forte

- Mínimo de 8 caracteres
- Pelo menos 1 número

O programa solicita senhas até que uma válida seja inserida ou o usuário digite 'sair'.

## Como executar

```bash
python -m src.main
```
